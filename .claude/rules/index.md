# ルールインデックス

作業ごとの参照先を示します。詳細な手順は各ファイルを参照してください。

| 作業 | 参照先 |
|---|---|
| **プロジェクト前提を確認する** | `.claude/rules/project-premises.md` |
| 静的モック制作の考え方を確認する | `.claude/rules/static-mockup-workflow.md` |
| Codex 委任の原則を確認する | `.claude/rules/codex-workflow.md` |
| 実装レビューを行う | `.claude/rules/implementation-review.md` |
| 実装委任を実行する | `.claude/commands/codex-exec.md`（`/codex-exec` Command） |
| mobile モック制作の方針を確認する | `.claude/rules/mobile-mockup.md`（mobile 作業時のみ自動ロード） |
| Docker / nginx 確認環境の方針を確認する | `.claude/rules/docker-preview.md`（Docker 作業時のみ自動ロード） |

## 注意

- 詳細な実行手順は `.claude/commands/codex-exec.md` が正本。
- 必要なルールだけを参照し、無関係なファイルを広げない。
